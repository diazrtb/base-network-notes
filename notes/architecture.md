# Base Architecture

## Overview

Base is built using the OP Stack, making it compatible with Ethereum while improving scalability.

## Main Components

- Execution Layer
- Settlement on Ethereum
- Data Availability
- OP Stack infrastructure

## Goal

The architecture is designed to support secure, scalable, and low-cost onchain applications.

## Benefits of the OP Stack

- Modular architecture
- Easy integration with Ethereum tools
- Improved scalability
- Lower transaction costs
- Open-source development model

## Notes

The OP Stack allows Layer 2 networks to share common infrastructure while maintaining compatibility with Ethereum.
